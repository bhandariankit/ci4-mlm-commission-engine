# CI4 MLM Commission Engine

Production-grade MLM commission engine built with CodeIgniter 4.

Designed for scalable MLM platforms with ROI systems, referral commissions, level income, binary/unilevel plans, withdrawal management, and strong admin payout control.

Built for real business operations with maintainable architecture, not demo-level MLM scripts.

# Database Design Philosophy

Built using normalized scalable architecture:

* separate earnings tables
* dedicated ROI level tables
* proper level progression logic
* payout history tracking
* wallet transaction separation
* commission source tracking
* audit-safe earning records

No messy payout calculations inside random controllers.
